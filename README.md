# Mask RCNN for Grass Detection

This project uses Mask RCNN to detect grass in images. The model is trained on a custom dataset and can be used for inference on new images.

## Requirements

- Python 3.6+
- TensorFlow 1.x
- Keras
- imgaug
- numpy
- scikit-image
- matplotlib
- Cython
- OpenCV
